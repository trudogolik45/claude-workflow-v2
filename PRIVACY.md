# Privacy Policy

This repository publishes a Claude Code plugin and related plugin resources for local installation in Claude Code.

## Data Handling

- The plugin bundle in this repository does not include a hosted backend or telemetry service.
- Any data access performed by installed skills, hooks, or local scripts happens in the user's own Claude Code session and local environment.
- Hook scripts operate only on the local repository checkout unless a user explicitly runs commands that access external services.

## Credentials

- Secrets are expected to be supplied by the user through local environment configuration.
- This repository should not be used to store production secrets or personal access tokens.

## Contact

For privacy questions, open an issue in this repository.
